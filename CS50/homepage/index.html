Course description

This is CS50x, Harvard University's introduction to the intellectual endeavors of computer science and the art of programming, for both majors and non-majors, with or without prior programming experience.

A level course Basic taught by David J. Malan, CS50x teaches students how to think algorithmically and solve problems efficiently. Topics include abstraction, algorithms, data structures, encapsulation, resource management, security, software engineering, and web development. Languages include C, PHP and JavaScript, as well as SQL, CSS and HTML. Problem sets inspired by real-world domains of biology, cryptography, finance, forensics, and gaming.
